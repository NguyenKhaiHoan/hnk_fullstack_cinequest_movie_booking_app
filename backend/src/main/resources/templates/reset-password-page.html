About CineQuest

CineQuest is a movie ticket booking app designed to simulate the movie booking experience. The backend is built using Java and Spring Boot, ensuring robust performance and scalability. The frontend is developed with Dart and Flutter, offering a smooth and responsive user interface.

The mobile UI design draws inspiration from Behance - Cinequest to create an engaging user experience. Additionally, Cinequest leverages the TheMovieDb (TMDB) API to retrieve movie data, enhancing the app with real-time movie information.

This app is developed solely for learning and personal development purposes.
